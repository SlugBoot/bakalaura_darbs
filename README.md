# Virtuālās laboratorijas izveide ar tīmekļa tehnoloģijām
Development of a virtual laboratory using web technologies


# Bakalaura darba mērķis
Izveidot mācību platformu tīmekļa pārlūkprogrammā, kurā ir iespējams apgūt un pilnveidot zināšanas par Linux sistēmu un citiem rīkiem virtuālā laboratorijas vidē, lai pasniedzējs varētu izveidot un novērtēt dažādus uzdevumus izmantojot virtuālas ierīces, kā arī lai students varētu šos uzdevumus izpildīt sev parocīgā vietā un vidē.
